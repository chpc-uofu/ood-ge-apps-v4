This Open OnDemand (OOD) app launches a per-user CryoSPARC instance on the HPC cluster, which you can access from your local web browser.
Both CryoSPARC Master and Worker run within the same Slurm job on a GPU compute node.

A CryoSPARC License ID is required. You can request an individual academic license ID at cryosparc.com/download.

Each user can have only ONE (1) active Cryosparc instance at any time.

The CryoSPARC database will be stored in the location you choose as the "Database Root", on a per-app-version basis.
When you start a new instance of the same version later, the previously stored database will be reused.
CryoSPARC databases may grow quickly, so choose a location with enough available space.
If the Database Root becomes full and writes are blocked, the database may become corrupted.
Note that the free-tier HOME space is capped at 50G/user.

Access to the launched CryoSPARC instance is protected by both CHPC login and CryoSPARC's native login.
After signing in with uNID credentials, you will be directed to CryoSPARC's login page.
Your CryoSPARC native username is fixed as <uNID>@cryo.edu. You can set a password in the OOD UI, or use the default password, which is the first 4 characters of the License ID.

You will receive Slurm email notifications when your instance reaches 80% and 90% of its walltime.
We recommend saving your work and manually shutting down the instance before walltime.
